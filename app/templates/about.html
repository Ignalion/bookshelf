Hi, my name is Constantine Panfilov. I'm python developer.
This is a bookshelf - test app made for prom.ua. It contains authorisation, a couple of pages where you can add authors or books, a simple search engine - you can search by book title or author's name. Books and authors has Many-To-Many relationship so you can link them between each other. Give it a try, make some fun.

Used technologies:
  • Flask
  • SQLAlchemy(Declarative)
  • SQLite RDBMS (Postgres on Heroku)
  • Alembic migrate tool
  • Jinja2 Templates
  • WTForms
  • Flask-Bootstrap
